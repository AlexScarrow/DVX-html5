# Development Log (DVX)

Use this as a quick continuity log between sessions.

---

## Template (copy for each session)

### Date
- YYYY-MM-DD

### Branch
- e.g. `main` or `feature/<name>`

### Goal
- Short statement of what this session aimed to do.

### Changes made
- File paths edited
- Key gameplay/visual logic changed
- Any constants tuned

### Test results
- What was tested in-game
- What worked
- What did not work

### Banked
- Commit hash + message
- Pushed? (yes/no)

### Known issues / risks
- Current bugs, edge cases, or unresolved concerns

### Next step
- Single concrete next action
